Kali Linux ve diğer Linux dağıtımları için önce terminalde sistemi güncellemek için



--------------------------------------------------
sudo apt update && sudo apt upgrade -y



--------------------------------------------------



Python ve pip’i yüklemek için:




----------------------------------------
sudo apt install python3 python3-pip -y
-








----------------------------------------

-------------------------------------------------------------------------
